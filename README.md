# 🎯 Project Overview

The objective of this project was to develop a **professional company website** that allows businesses to:

- showcase services and training programs
- capture potential leads through forms
- communicate with visitors instantly
- provide downloadable course information
- interact with visitors through chatbot automation

The website is optimized for **desktop, tablet, and mobile devices**.

---

# ⚙ Workflow (Lead Capture Process)


User visits website
│
▼
User fills contact form
│
▼
Form data processed
│
▼
Email notification triggered
│
▼
Company receives inquiry instantly


This ensures **fast communication between visitors and the company**.
